Friendly Frame

#typescript #angular #aws
  • Project landing page friendlyfra.me

This is a personal (and massive) project of mine that I have been working on for a while now. The aim of the project is to make it easier for any business to manage multiple remote informational displays from a central platform. The remote screens can show anything, from digital store signage to business metric displays - Friendly Frame is simply a platform to manage and monitor these devices.

Although I have engineered the entire platform stack, I would say that I most enjoy working on the backend infrastructure of this project. In a nutshell, it is a Node app written in TypeScript that runs a REST API consumed by the rest of the platform. The backend currently runs as an auto-scaling Elastic Beanstalk application on AWS, but in the future I would love to migrate the entire app to run serverless using AWS Lambda functions. I also plan on looking into Terraform to generate the platform's AWS infrastructure as I would love to have this under source control.
